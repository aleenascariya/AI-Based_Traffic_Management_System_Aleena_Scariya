# NeuralTraffic AI

NeuralTraffic AI is a smart traffic management dashboard built with React and TypeScript.

## Features

- Adaptive and fixed traffic control modes
- Vehicle injection and lane simulation
- Dynamic signal override controls
- AI recommendations and insights
- Traffic statistics and health monitoring
- Activity logging
- Traffic record storage

## Technologies

- React
- TypeScript
- Vite
- CSS
